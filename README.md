# esp32-serial-display-adc

ESP32 com serial, LCD I2C e leitura ADC — firmware final do PSEE.

| Byte | Função |
|------|--------|
| `'Y'` | Início de frame |
| `'W'` | Fim + exibe no LCD |
| `'X'` | Leitura ADC → tensão |

**Baud rate:** 9600
